\# ChatApp



ChatApp is a real-time anonymous chat room application built using ASP.NET Core and SignalR. It allows users to create temporary chat rooms, join using room codes or links, and communicate instantly without requiring login or permanent accounts.



\## Project Overview



This project focuses on temporary and privacy-friendly communication. Users can create listed or private rooms, join anonymously, and send messages in real time. The application is designed with a layered architecture to separate UI, business logic, infrastructure, and testing.



\## Features



\- Anonymous chat without user registration

\- Temporary chat rooms

\- Listed and private room options

\- Join rooms using room code

\- Real-time messaging using SignalR

\- Session-based anonymous user handling

\- Admin and health check pages

\- Clean layered project structure

\- Unit tests for chat flow validation



\## Tech Stack



\- ASP.NET Core

\- Blazor

\- SignalR

\- C#

\- .NET

\- HTML/CSS

\- xUnit for testing
